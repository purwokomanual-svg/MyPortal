Import Data Penjualan (CSV)
📂

Seret & lepas file CSV di sini

atau klik untuk pilih file · CSV dari Shopee / Tokopedia / TikTok Shop

Format CSV:

no_pesanan, tanggal, marketplace, produk, varian, kategori, qty, total, status, biaya_admin, biaya_tambahan

Kolom biaya_admin dan biaya_tambahan opsional (Rp) — jika kosong, akan dihitung estimasi otomatis dan bisa diisi manual lewat Edit Pesanan.

🔄 Rekonsiliasi Stok ⇄ Penjualan

Jika angka "terjual" di menu Stok Gudang tidak sesuai dengan jumlah sebenarnya di Data Penjualan (misalnya karena data pernah diimport dari CSV sebelum perbaikan ini, atau nama produk sempat tidak persis sama), klik tombol di bawah untuk menghitung ulang stok dan terjual semua SKU dari awal berdasarkan seluruh data Penjualan yang aktif (bukan "Dibatalkan").

👥 Manajemen User & Hak Akses
Atur siapa saja yang boleh masuk ke dashboard ini dan apa yang boleh mereka lakukan. Owner = akses penuh termasuk pengaturan & user. Staff = bisa kelola pesanan/stok, tidak bisa ubah pengaturan.
